# 版本路线图

 我们会不停的改进52ABP-PRO。这里说功能列表将在下一个版本中实现：



 - 阿里云 / SMS短信服务
 - 阿里云 / VOD视频点播
 - 阿里云 / OSS对象存储
 - 微信 / 开放平台（快捷登陆）
 - 微信 / 公众平台（身份授权）
 - 微信 / 微信支付（商户平台）
 - QQ / QQ互联（快捷登陆）
 - 微信SDK集成
 - 微信 / 公众平台（身份授权）
