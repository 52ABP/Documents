
# 入门

## 项目技术说明

> 项目基础类库使用介绍，文档整合，帮助开发者快速上手！

> how to run ?  [文档地址](https://docsify.js.org/#/zh-cn/quickstart)

- 使用vscode 打开，在终端上输入以下命令

* 安装docsify-cli命令
```
    npm i docsify-cli -g
```
定位到docs目录下可执行此命令
```bash
    docsify serve .
```
或直接在docs上一级目录下可执行此命令
```
 docsify serve docs
```

最后打开  [http://localhost:3000](http://localhost:3000)
